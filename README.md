# Civilization VI Guide

这是一个用于《Sid Meier's Civilization VI / 文明 6》的中文答疑、攻略、新手教程和资料索引项目。

当前默认语境：

- 游戏版本：文明 6 标准版
- DLC/资料片：无
- 默认规则集：标准规则
- 沟通语言：简体中文
- 资料优先级：官方手册优先，其次文明百科/社区百科，再其次社区攻略和 Steam 创意工坊

## 如何给 AI 使用

让 AI 先读：

1. `AI_CONTEXT.md`
2. `civilization-vi-guide/SKILL.md`
3. `civilization-vi-guide/references/source-map.md`
4. `civilization-vi-guide/references/manual-index.md`

如果问题是操作、UI、快捷键、基础机制，优先查官方手册索引。如果问题是底层数值、隐藏机制、具体条目，优先用官方手册确定概念，再查文明百科或灰机 wiki。
