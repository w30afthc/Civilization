# Workshop Mods

Use this file for Steam Workshop items the user may consider later. The user's normal game context is Civilization VI Standard Edition with no DLC, so check each mod for base-game compatibility before recommending installation.

## General Rules

- Prefer UI/QoL mods over gameplay-changing mods for a first mod setup.
- Do not recommend subscribing to a whole collection blindly. Check dependencies, DLC requirements, compatibility notes, last update date, and comments.
- Avoid mods that change rules, leaders, civilizations, yields, AI, or balance unless the user explicitly wants a modded run.
- If a mod says it requires Rise and Fall, Gathering Storm, New Frontier Pass, or Leader Pass, mark it as not suitable for the user's current standard-only setup.
- For ongoing saves, prefer mods that clearly state they are save-compatible; otherwise install before starting a new game.

## District Removal Mods

### Removable Districts

- Link: https://steamcommunity.com/sharedfiles/filedetails/?id=970184582
- Title: Removable Districts
- Type shown by Steam: Gameplay mod.
- Posted/updated shown by page: posted 2017-07-07, updated 2020-03-31.
- What it claims to do: adds city projects that can remove districts after that district has been built.
- Cost note from page: removal projects cost half the normal district build cost.
- Limitations from page: wonders cannot be removed; removing neighborhoods may not let the player choose exactly which one is removed.
- Important caution: Steam currently shows warnings that the item has been removed from the community and is incompatible with Civilization VI. Recent comments also mention broken behavior and some unsupported districts. Treat as risky and verify before recommending.
- Not a building-removal solution: this is about districts. It does not provide a general safe way to delete individual completed buildings.
