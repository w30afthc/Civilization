## Civilopedia Rule Sets

Default to the standard-rules URL for this user's normal gameplay. Choose a different rule-set URL only when the user explicitly mentions DLC, a screenshot shows DLC-only mechanics, or the question compares DLC systems.

- **Standard rules / 标准规则**: base rules. Use for users who do not own or do not enable expansions.
- **Rise and Fall / 资料片：迭起兴衰**: use when the user enables Rise and Fall but not Gathering Storm, especially for loyalty, governors, era score, golden/dark ages, and related civilization/leader changes.
- **Gathering Storm / 资料片：风云变幻**: use when the user enables Gathering Storm. It includes many Rise and Fall systems and adds or changes disaster, climate, power, strategic resource, grievance, diplomatic favor, World Congress, and late-game diplomacy behavior.

If the user does not say which DLC is active, assume standard rules for this project. Do not ask about DLC for ordinary gameplay questions unless the answer would be wrong under standard rules.

## What Each Source Is For

### Official Manuals

Use the local official manuals first for standard-rules fundamentals, UI, keyboard shortcuts, and official conceptual explanations. Read `manual-index.md` first for page references, then inspect the Traditional Chinese manual and cross-check against the English manual when wording is unclear.

The manuals are 2016 standard-rules references. Do not use them for DLC-only systems.

### Civilopedia

Use for rule lookups and exact terminology. Its navigation covers concepts, civilizations/leaders, city-states, districts, buildings, wonders/projects, units, promotions, great people, technologies, civics, governments/policies, religion, terrain/features, resources, improvements, and routes.

It separates rule sets: standard rules, Rise and Fall, and Gathering Storm. Always check the selected rule set when answering questions about loyalty, governors, era score, disasters, power, climate, diplomatic favor, World Congress, grievances, strategic resources, or late-game systems.

### Huiji Wiki Community Encyclopedia

Use the Huiji wiki for detailed community-maintained data, hidden mechanics, tables, and strategy context. Treat it as secondary to official manuals for basic rule wording. If browsing the site is blocked, say so and use Civilopedia/manuals plus general knowledge.

### Wikipedia

Use for high-level explanations of Civilization VI as a 4X turn-based strategy game, the broad changes from prior entries, and general concepts such as districts, technology tree, civics tree, eurekas/inspirations, wonders, and platforms/release context.

Do not use Wikipedia as the only source for exact in-game numbers.

### Steam Community Guides

Use for community guides, walkthroughs, achievements, leader-specific advice, mod setup, and troubleshooting. Treat advice as community opinion unless corroborated by in-game data or multiple guides.

Useful browsing filters include Gameplay Basics, Walkthroughs, Achievements, Characters, Modding or Configuration, Multiplayer, and Most Popular/Top Rated/Most Recent.

### Steam Workshop

Use Workshop links for optional mods only. Read `mods.md` first for the user's saved candidates and compatibility cautions. Since the user currently has the standard edition with no DLC, check whether each mod supports vanilla/base game before recommending it.

### Zhihu Guide Collection

Use as a user-provided external guide index if accessible. If access is blocked or incomplete, say so and rely on accessible sources plus general game knowledge.

Do not reproduce long text or images from the guide collection. Summarize only what is necessary and link back.

## Source Selection Rules

- For "这个机制怎么算": Civilopedia first, then explain in plain Chinese.
- For "快捷键/界面按钮/基础操作": local official manuals first, especially `manual-index.md`.
- For "底层数据/精确数值/隐藏机制": official manuals for concept, then Huiji wiki or Civilopedia for details.
- For "新手怎么玩": bundled playbook first, Civilopedia only for terms.
- For "某领袖怎么打": Civilopedia for abilities and unique items, Steam/Zhihu for strategy opinions.
- For "忠诚/总督/时代分": explain that these are not in standard rules and require Rise and Fall or Gathering Storm.
- For "灾害/气候/电力/外交支持/世界议会/怨气": explain that these are not in standard rules and require Gathering Storm.
- For "这个 mod/创意工坊/启动问题": Steam guides and current browsing first.
- For "装哪些 UI mod/这个合集能不能用": read `mods.md`, then browse the current Workshop page if compatibility or last update matters.
- For "最新版本/当前社区排行": browse; do not rely on memory.
